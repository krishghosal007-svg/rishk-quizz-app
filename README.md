# rishk-quizz-app
# 🧠 RISHK QUIZ – Dynamic Quiz Web Application

A modern and interactive **Quiz Web Application** built using **HTML, CSS, and JavaScript**.
This app allows users to **sign up, log in, choose quiz categories, answer timed questions, and track their performance** with statistics, achievements, and marksheets.

The project features a **professional UI with animations, responsive design, and interactive elements** to provide a smooth quiz experience.

---

## 🚀 Features

✨ **User Authentication**

* Login and Signup system
* Username-based authentication
* Logout functionality

📚 **Quiz Categories**

* Browse multiple quiz categories
* Search categories instantly
* Dynamic category loading

⏱ **Timed Questions**

* Countdown timer for each question
* Progress bar tracking quiz progress
* Score updates in real-time

📊 **Dashboard & Analytics**

* Total quizzes attempted
* Total score tracking
* Average accuracy calculation

🏆 **Achievements System**

* Earn badges for quiz performance
* Achievement progress tracking

📄 **Marksheet System**

* View past quiz attempts
* Date, category, score, and result status

🎨 **Modern UI**

* Glassmorphism design
* Animated background
* Smooth transitions
* Responsive layout

## 🧪 How It Works

1. User **creates an account or logs in**
2. Dashboard displays **statistics and achievements**
3. User selects a **quiz category**
4. Quiz starts with **timed questions**
5. Score is calculated instantly
6. Results are stored in the **marksheet system**
